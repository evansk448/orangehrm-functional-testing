# OrangeHRM Manual Functional Testing Project

## Overview
Performed manual functional testing on the OrangeHRM demo web application, focusing on core user workflows, input validation, and system behavior across multiple modules.

## Scope
Modules tested:
- Login
- Forgot Password
- Recruitment
- Directory
- PIM (Employee Management)
- Time (Attendance)

## Objectives
- Validate core application workflows
- Verify input validation and error handling
- Ensure accurate system responses to user actions
- Identify defects impacting usability and functionality

## Skills Demonstrated
- Test Case Design
- Functional Testing
- Negative Testing
- Input Validation
- Workflow Analysis
- Defect Documentation

- Test_Cases.md
- Defects/

OrangeHRM Demo Application

## Summary
This project demonstrates the ability to analyze a web-based application, design test scenarios across multiple modules, and document defects based on system behavior and user interaction.
